## Step 1: Authenticate

Get an access token using your credentials.

<CodeGroup>

```bash cURL
curl -X POST 'https://core-dev.quidkey.com/api/v1/oauth2/token' \
  -H 'Content-Type: application/json' \
  -d '{
    "client_id": "your-client-id",
    "client_secret": "your-client-secret"
  }'
```

```javascript Node.js
const response = await fetch('https://core-dev.quidkey.com/api/v1/oauth2/token', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    client_id: 'your-client-id',
    client_secret: 'your-client-secret'
  })
});

const { access_token } = await response.json();
console.log('Access token:', access_token);
```

```python Python
import requests

response = requests.post(
    'https://core-dev.quidkey.com/api/v1/oauth2/token',
    json={
        'client_id': 'your-client-id',
        'client_secret': 'your-client-secret'
    }
)

access_token = response.json()['data']['access_token']
print(f'Access token: {access_token}')
```

</CodeGroup>

<Check>
You should receive an `access_token` valid for 15 minutes. Save this for the next step.
</Check>

<Tip>
Don't have credentials yet? [Try it in the API playground](/api-reference/auth/issue-a-new-access-token) first - no authentication required for testing!
</Tip>

## Step 2: Create a Payment Request

Use your access token to create a payment request.

<CodeGroup>

```bash cURL
curl -X POST 'https://core-dev.quidkey.com/api/v1/embedded/payment-requests' \
  -H 'Authorization: Bearer YOUR_ACCESS_TOKEN' \
  -H 'Content-Type: application/json' \
  -d '{
    "customer": {
      "name": "John Doe",
      "email": "john@example.com",
      "phone": "+4917646793347",
      "country": "DE"
    },
    "order": {
      "order_id": "test-order-001",
      "amount": 1000,
      "currency": "EUR",
      "payment_reference": "Test Payment",
      "locale": "en-GB",
      "test_transaction": true
    },
    "redirect_urls": {
      "success_url": "https://yoursite.com/success",
      "failure_url": "https://yoursite.com/failure"
    }
  }'
```

```javascript Node.js
const response = await fetch('https://core-dev.quidkey.com/api/v1/embedded/payment-requests', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${access_token}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    customer: {
      name: 'John Doe',
      email: 'john@example.com',
      phone: '+4917646793347',
      country: 'DE'
    },
    order: {
      order_id: 'test-order-001',
      amount: 1000, // â‚¬10.00 in cents
      currency: 'EUR',
      payment_reference: 'Test Payment',
      locale: 'en-GB',
      test_transaction: true
    },
    redirect_urls: {
      success_url: 'https://yoursite.com/success',
      failure_url: 'https://yoursite.com/failure'
    }
  })
});

const { payment_token } = await response.json();
console.log('Payment token:', payment_token);
```

</CodeGroup>

<Check>
You should receive a `payment_token` - save this for embedding in the iframe.
</Check>

<Note>
**Amount format:** Use cents (minor units). `1000` = â‚¬10.00. Same format as Stripe!
</Note>

## Step 3: Embed the Checkout

Add the Quidkey iframe to your checkout page using the `payment_token` from Step 2. Quidkey will automatically predict and pre-select the customer's bank based on their information, so they can pay with their trusted bank brand.

```html
<!DOCTYPE html>
<html>
<head>
  <title>Quidkey Test Checkout</title>
  <style>
    .quidkey-container {
      width: 100%;
      max-width: 500px;
      margin: 50px auto;
    }
  </style>
</head>
<body>
  <div class="quidkey-container">
    <h1>Test Payment - â‚¬10.00</h1>

    <iframe
      id="quidkey-iframe"
      src="https://core-dev.quidkey.com/api/v1/embedded?payment_token=YOUR_PAYMENT_TOKEN"
      style="width:100%;height:108px;border:0;"
      title="Quidkey checkout"
    ></iframe>

    <button id="pay-button" disabled>Pay â‚¬10.00</button>
  </div>

  <script>
    let selectedBankId = null;

    // Listen for bank selection from iframe
    // Quidkey predicts and pre-selects the customer's bank automatically
    window.addEventListener('message', (event) => {
      if (event.data.type === 'quidkey-state-update') {
        selectedBankId = event.data.selectedBankId;

        // Enable button when bank is pre-selected or selected
        document.getElementById('pay-button').disabled = !selectedBankId;
      }
    });

    // Handle payment initiation
    document.getElementById('pay-button').addEventListener('click', async () => {
      // Call your backend to initiate payment
      const response = await fetch('/api/quidkey/initiate', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
          bankId: selectedBankId,
          paymentToken: 'YOUR_PAYMENT_TOKEN'
        })
      });

      const { payment_link } = await response.json();
      window.location.href = payment_link; // Redirect to bank
    });
  </script>
</body>
</html>
```
